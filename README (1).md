# 🧍‍♂️ Stand Duel System

Simulador de batalhas 1x1 entre *Stands*, inspirado no universo de *JoJo's Bizarre Adventure*.

Este projeto foi desenvolvido por **Vinícius Custódio Chelli**, utilizando **Java** com foco em Programação Orientada a Objetos, lógica de combate baseada em atributos de Stands e efeitos especiais por meio de habilidades únicas.

---

## ✨ Funcionalidades

- 🥊 Simulação de batalhas entre dois Stands.
- ⚔️ Sistema baseado em atributos (Destruição, Velocidade, Alcance, etc.).
- 🔁 Batalhas por turnos com até 10 rodadas.
- 💥 Habilidades especiais únicas para cada Stand.
- 🗣️ Finalização com Stand Cry!

---

## 📦 Como usar

### Requisitos:
- Java JDK 11 ou superior instalado
- Terminal ou IDE compatível com Java

### Execução:
```bash
javac StandDuel.java
java StandDuel
```

## ⚙️ Complexidade

- **Tempo de Execução:** `O(n)` com `n = número de turnos` (máximo 10)
- **Espaço:** `O(k)` com `k = número de Stands carregados`
